# 角色设计指南

## 角色设计流程

### 1. 角色概念定义
- **核心一句话：** 用一句话定义这个角色
- **故事角色：** 主角/配角/反派/功能性角色
- **情感弧光：** 角色在故事中的变化

### 2. 外观设计

#### 面部特征
```
脸型: 圆脸/方脸/鹅蛋脸/长脸
眼睛: 大眼/细眼/凤眼/杏眼 + 颜色
鼻子: 挺拔/小巧/鹰钩
嘴巴: 薄唇/厚唇/樱桃小嘴
皮肤: 白皙/小麦/古铜/深色
标志性特征: 痣/疤痕/纹身/胎记
```

#### 发型
```
长度: 短发/中长/长发
样式: 直发/卷发/波浪/脏辫
颜色: 自然色/挑染/全染
标志性: 特殊发饰/扎法
```

#### 体型
```
身高: 矮小/中等/高大
体型: 纤细/匀称/壮硕/圆润
姿态: 挺拔/佝偻/随意
```

### 3. 服装系统

为每个角色设计至少 3 套服装：

| 套装 | 场景 | 设计要点 |
|------|------|----------|
| 日常 | 家居、休闲 | 舒适、体现性格 |
| 工作/战斗 | 职业/冒险 | 功能性、专业感 |
| 特殊 | 仪式、重要场合 | 正式感、故事意义 |

### 4. 表情系统

标准表情集（至少 8 个）：
```
😊 微笑 — 嘴角上扬，眼睛微眯
😠 愤怒 — 眉毛下压，嘴唇紧抿
😢 悲伤 — 眉毛上挑，嘴角下垂
😲 惊讶 — 眉毛上扬，嘴巴微张
😨 恐惧 — 瞳孔放大，面色苍白
🤔 思考 — 手托下巴，目光凝视
😴 困倦 — 眼睛半闭，打哈欠
😏 得意 — 单边嘴角上扬
```

### 5. AI 生成一致性控制

#### 关键原则
1. **固定基础描述** — 每次生成使用相同的基础描述
2. **只改变量** — 变化部分（表情、服装、场景）单独修改
3. **使用参考图** — 有参考图时一致性最高
4. **描述词排序** — 最重要的特征放在最前面

#### 角色描述卡模板
```
角色名: [Name]
基础提示词: "A [age] [gender] with [hair color] [hair style], [eye description], [face shape], [skin tone], [build], [distinctive feature]"
表情前缀: 根据表情替换 "with a [表情] expression"
服装后缀: "wearing [服装描述]"
```
