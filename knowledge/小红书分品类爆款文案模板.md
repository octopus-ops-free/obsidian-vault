## 一、6大通用爆款笔记结构

### 1. 种草测评型（变现能力最强）
**标题：** 产品名 + 使用效果/核心卖点
**结构：**
- 开头：个人背景/痛点引出（1-2句话建立共鸣）
- 产品介绍：品牌、价格、规格
- 使用体验：分维度描述（质地、效果、持久度）
- 优缺点总结：客观列出，增强可信度
- 结尾：适合人群推荐 + 互动引导

### 2. 干货合集型（爆文最多，收藏率最高）
**标题：** 数字 + 核心内容 + 结果/价值
**结构：**
- 开头：痛点共鸣或场景引入
- 正文：分点罗列（3-10个要点，每点配简短说明）
- 结尾：总结一句话 + 收藏引导
**要点：** 适当增加数字，用限定词区分（按季节/肤质/价格/人群）

### 3. 避坑排雷型（点击率高）
**标题：** "避坑"/"别买"/"踩雷" + 具体品类
**结构：**
- 开头：踩坑经历，制造紧张感
- 正文：逐一列出坑点，每个坑配解决方案
- 结尾：正确做法总结 + 互动引导

### 4. 教程攻略型（收藏率高）
**标题：** "手把手教你"/"保姆级教程" + 具体内容
**结构：**
- 开头：最终效果展示（先给读者预期）
- 正文：Step 1/2/3... 分步骤详细讲解
- 结尾：注意事项 + 互动引导

### 5. 对比测评型（信任感最强）
**标题：** A vs B + 核心差异点
**结构：**
- 开头：选择困难的场景引入
- 正文：从多个维度逐一对比（价格、效果、使用感等）
- 结尾：给出明确建议 + 引导评论区讨论

### 6. 个人故事型（转发率最高）
**标题：** 时间线/转折 + 结果
**结构：**
- 开头："before"状态（引发共鸣）
- 转折：触发改变的契机
- 过程：具体做了什么（干货输出）
- 结果："after"状态（激励读者）
- 结尾：鼓励互动

## 二、美妆护肤类

### 爆款标题公式
1. **肤质+效果** — "黄皮必入！这支口红显白到离谱"
2. **人群+痛点** — "油痘肌早八持妆8小时不卡粉！3款百元内亲测"
3. **前后对比** — "从满脸痘印到素颜发光，我只用了这3步"
4. **情绪夸张** — "救命！这个玻尿酸精华太绝了！"
5. **数字效果** — "坚持用这瓶美白精华一个月，同事问我是不是打了水光针"

### 5种高收藏内容类型
1. **自用分享型** — "无广分享"、"自用"关键词，真实感强
2. **新品测评型** — 新品诱惑 + 视觉观感 + 使用感受
3. **产品合集型** — 按肤质/价格/季节分类，信息密度高
4. **怼脸对比型** — 妆前妆后反差越大越好，视觉冲击力
5. **单品种草型** — 提取核心卖点 + 高网感文案 + 情绪化压字

### 正文结构模板（美妆专用）
```
肤质说明 → 产品介绍 → 上脸效果 → 优缺点 → 适合人群
```

### 高转化文案句式
- "救命！这款XX简直是YY，用完ZZ"
- "姐妹们听劝！XX就认准它，[效果描述]"
- "别再拿XX硬撑了！这个YY是[人群]的后悔药"
- "挖到宝了！这个XX是YY救星，上脸瞬间[效果]"
- "空瓶了X个！每次用都像[比喻]，[情绪词]满满"

### 护肤文案按需求分类
| 需求类型 | 文案重点 | 案例句式 |
|---------|---------|---------|
| 熬夜急救 | 即时效果+状态恢复 | "敷完第二天开会自带高光" |
| 保湿维稳 | 质地+持久度+肤感 | "沙漠皮也能hold住一整天" |
| 美白提亮 | 时间线+肉眼可见变化 | "一个月同事问我是不是打了水光针" |
| 抗老紧致 | 年龄焦虑+效果可见 | "脸部轮廓都清晰了，法令纹也变浅了" |
| 成分党 | 专业+温和+有效 | "高浓度玻色因，用一次就爱上" |

## 三、美食类

### 爆款标题公式
1. **场景+特点+情绪** — "一人食晚餐｜10分钟搞定，好吃到哭"
2. **教程型** — "0失败教程！在家复刻茶颜悦色同款奶茶"
3. **合集型** — "成都本地人私藏的10家小馆子，第3家绝了"
4. **健康减脂** — "减脂期也能吃的5道神仙甜品｜低卡又解馋"

### 正文结构
```
食材清单 → 步骤教学（编号+emoji） → 成品展示 → 口感描述 → 小贴士
```

### 高收藏要素
- 步骤清晰，每步配图
- 精确用量（克数、勺数）
- 常见失败原因+解决办法
- "收藏=学会" 的信息密度感

## 四、穿搭类

### 爆款标题公式
1. **身材痛点+解决方案** — "微胖女生这样穿，显瘦20斤"
2. **价格+惊喜感** — "100块搞定职场穿搭！上班族也能优雅通勤"
3. **季节+风格** — "秋冬最温柔的5套穿搭｜回头率200%"
4. **身高限定** — "155cm小个子｜这样穿腿长1米8"

### 正文结构
```
身材说明（身高体重） → 单品清单（品牌+价格） → 搭配思路 → 穿搭效果 → 购买链接
```

### 高收藏要素
- 真人上身图（非模特图）
- 明确标注身高体重
- 按场景分类（通勤/约会/旅行）
- 提供平价替代方案

## 十、AI辅助写文案的正确姿势

1. **AI生成初稿** — 输入关键词+品类+目标人群，生成框架
2. **人工润色** — 加入真实体验、个人语气、细节描写
3. **违禁词检测** — 发布前检查极限词、敏感词
4. **A/B测试** — 同一内容用不同标题测试效果

### 关键原则
- 模板是框架，不是限制
- 同一个模板要有变化（换开头方式、调整段落顺序）
- 真实内容 > 完美模板
- 数据显示：使用爆款模板的笔记平均互动率比无模板笔记高出260%
